# Kycek Studios - Premium Website

Premium e-commerce website for Kycek Studios, inspired by nude-project.com.

## Features
- Premium design with extreme attention to detail
- Bundle options with 20% discount
- Email capture for 15% off
- Limited edition sold out product with waitlist
- 4 SKUs: Essential Tee (Black/White), Training Short (Black/White)
- Mobile-first responsive design

## Tech Stack
- HTML5, CSS3, JavaScript
- No frameworks for maximum performance
- Custom design system

## Pages
1. Homepage
2. Product pages (4 SKUs)
3. Limited Edition (Sold Out)
4. Collections
5. About
6. Contact

## Conversion Features
- Bundle CTAs
- Email popup (15% off)
- Scarcity indicators
- Social proof elements
